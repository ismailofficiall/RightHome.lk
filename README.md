# Estate Agent Web App - 5COSC026W
**Mohammed Ismail Ibrahim / 20233113**

## Features
* **Search:** Filter by type, price, bedrooms, date, and postcode.
* **Favourites:** Drag and drop or click to add properties to your favourites.
* **Property Details:** View image galleries, floor plans, and Google Maps tabs.

## How to Run
1. `npm install`
2. `npm start`
3. Run tests with: `npm test`
